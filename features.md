OpenCanvas Features List
This document outlines the key features of the OpenCanvas application, categorized for the purpose of capturing screenshots and screen recordings.

1. Dashboard & Workflow Management
Goal: Showcase the organization and management capabilities.

Workflow Creation:
Create a blank new workflow.
Import workflow from JSON.
Workflow Organization:
Grid View: Visual cards with thumbnail previews.
List View: Detailed table view with sortable columns.
Search: Real-time filtering of workflows by name.
Workflow Actions:
Rename: Update workflow names via dialog.
Duplicate: Clone existing workflows.
Export: Download workflow as JSON.
Share: Generate shareable links.
Delete: Remove unwanted workflows.
2. The Visual Editor
Goal: Showcase the intuitive drag-and-drop interface.

Canvas Interactions:
Drag and drop nodes.
Connect nodes via handles.
Pan and Zoom navigation.
Context menus (Right-click node/canvas).
Node Management:
Node Library: Sidebar with categorized tools and models.
Copy/Paste: Duplicate nodes within the canvas.
Undo/Redo: History management for changes.
Sticky Notes: Add annotations directly to the canvas.
3. Generative AI Capabilities (Google Gemini)
Goal: Highlight the power of integrated AI models.

Text & Prompting
Prompt Node: Basic text input for instructions.
Prompt Enhancer: Automatically refine and improve prompts using AI.
Prompt Concatenator: Merge multiple text sources into a single prompt.
Image Generation & Analysis
Imagen 4.0: High-fidelity image generation node.
Gemini 3 Pro Image Preview: Fast image generation with optional Google Search grounding.
Image Describer: Analyze images and generate text descriptions (Image-to-Text).
Gemini 2.5 Flash Image: Efficient image processing and analysis.
Video Generation & Analysis
Veo 3.1 Generate Preview: AI video generation from prompts.
Video Describer: Analyze video content and generate text descriptions (Video-to-Text).
4. Media Processing Tools
Goal: Demonstrate the built-in media manipulation tools.

Image Editing
Image Compositor: Advanced layering, blending, and positioning of multiple images.
Painter: In-painting tool with masking capabilities for selective editing.
Crop: Precise image cropping/resizing.
Blur: Apply blur effects to images.
Color Grading: Adjust RGB channels and gamma levels.
Video Utilities
Video Upload: Direct video file ingestion.
Extract Video Frame: Capture specific frames from video for image processing.
5. Screen Recording Scenarios
Suggested flows for video demos:

"From Prompt to Image": Start with a prompt, enhance it, and generate an image using Imagen 4.0.
"Multimodal Magic": Upload a video, extract a frame, describe it, and use that description to generate a new variation.
"Creative Composition": Generate two images and blend them using the Image Compositor.
"Video Creation": Use a text prompt to generate a video clip with Veo 3.1.
